WEB & MOBILE PROGRAMMING

This is ICP2. In this module we are going to work on HTML, CSS. We are going to create a static web page and styles to it using CSS. The sequence of this module is

  • Create the web page using all elements.
  • Add styling to the webpage using CSS.
  • Create a static website of google home page using HTML and CSS.

    • The HTML attributes used in this module are:

  1. Block elemnts as div
  2. Inline elements as span
  3. Paragraph as p
  4. ordered Lists as ol
  5. Unordered lists as ul
  6. Image attributes as a and href
  7. Video attributes as source

Web designis important because it impacts how your audience perceives your brand. The impression you make on them can either get them to remain on your page and learn about your business or leave your page and turn to a competitor. A good web design helps you keep your leads on your page.
